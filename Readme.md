# HabiTrack

A simple no BS and no adverts app to track your habits.

## Features

- Add new habits
- Mark habits as completed
- Create Categories
- Other features coming soon

## Getting Started

### Dependencies

- React Native
- Expo
- React Navigation
- React Query
- Drizzle ORM
- Expo SQLite

### Installation

- `npm install`
- `npm run start` For expo go version
